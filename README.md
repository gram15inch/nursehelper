# nursehelper
밖에서 간호업무를 볼때 사용하는 어플입니다. </br></br>
매 일정마다 문서를 수기로 작성하고 따로 컴퓨터에 저장하는 수고를 덜기위하여 고안하였습니다. </br></br>
문서 작성과 더불어 일정까지 관리할수있게 하였습니다.</br>

## 기능
* 문서 관리
* 일정 관리
* 사용자 인증
