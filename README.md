# nursehelper
간단한 문서작성과 일정을 관리하는 앱입니다.

## 기능
* 문서 관리
* 일정 관리
* 사용자 인증

## 문서관리
* 문서 검색
  - 카테고리별 검색

   ![Animation](https://user-images.githubusercontent.com/94951889/147647251-af99c207-ff4c-4102-a6c5-6c430151577b.gif)

 - 문서탭에서 수정
 
* 문서 추가
 - 일정생성시 자동생성


## 일정관리
 * 일정 검색
 
 * 일정 추가
 
 * 일정 수정
 
 * 일정 삭제

## 사용자 인증
 * 회원가입/로그인/로그아웃
